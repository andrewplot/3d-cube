# 3D Rotating Cube

An interactive 3D cube with full rotation and translation controls.

## About

Originally prototyped in Desmos, then converted to HTML5 Canvas for a smoother interactive experience.
Desmos link: https://www.desmos.com/calculator/6l8gvtyafs

## Controls

- **Drag** the canvas to rotate
- **Sliders** for precise control
- **Auto-Rotate** button for animation
